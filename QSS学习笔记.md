# QSS学习笔记

```
background-color：背景颜色
border-radius：边界半径（圆角）
padding：自身边框到自身内部另一个容器边框之间的距离
layoutSpacing：布局中相邻组件的间隔
background:url();//平铺
background-image:url();//平铺
border-image：url();//拉伸填充
```

## QPushButton

```css
QPushButton{
	border-radius: 5px;
	border:2px solid rgba(228,228,228,1);
	padding:2px;
	background-color: rgb(255, 255, 255);
}
QPushButton::pressed{
	background-color: rgb(220, 220, 220);
}
```
